Traffic Dynamic Simulator

A C++ project demonstrating traffic flow with cars, queues, and traffic lights.

About the Project

A console-based traffic simulation that models cars, traffic lights, and pedestrian crossings using linked lists, queues, and stacks. The project includes a real-time clock, a color-coded UI, and save/load features to replicate real-world traffic management.
